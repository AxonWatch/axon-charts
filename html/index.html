Why Axon Charts?

Designed for quants, traders, and AI agents who need charts that work — instantly.

Native AI Discovery

Every chart registers in a global discovery layer. AI agents find them, read structured market data, and interact directly — no middleware needed.

Sub-Millisecond Rendering

Sub-millisecond tick updates. Zero dependencies, zero layout shifts, zero jank at any zoom level.

Multi-Chart Dashboards

Every chart gets a unique identifier. Agents target the right chart by name, ticker, or position. Commands land where they belong, even in dashboards with dozens of charts.

Data Control

By default agents see chart dimensions and price ranges. One toggle unlocks full bar-level price and volume. Toggle discovery off to make charts invisible to AI entirely.

Everything Configurable

Layout, scales, crosshair, colors, timezone, percentage mode, right-click menu, streaming — every aspect has an option and a default. Changes apply immediately.

TypeScript-Native Architecture

Full type definitions, predictable APIs, and a clean extension model. Add custom indicators, overlays, or analysis logic by extending base classes.

Built for Every Trading Style

From classic candlesticks to trend-smoothed Heiken-Ashi -- six series types, volume, multiple price scales, structured LLM context exports. All in one library.

Simple, Powerful API

Designed for human developers and AI agents alike. Every method is predictable. Every option has a default. Every chart has an ID.

Chart Creation

const chart = AxonCharts.createChart('#container', options)

Create a new chart instance with container selector and initial options. Returns a chart object with full API access.

Set Options

chart.setOptions({ volume: { show: true } })

Update any chart option dynamically. Changes are applied immediately without re-rendering the entire chart.

Load Data

chart.setData([{ time, open, high, low, close, volume }])

Load OHLCV data array. Automatically scales and renders. Supports timestamps in milliseconds.

Real-time Updates

chart.updateLastBar({ time, close, volume })

Update the latest candle efficiently. Perfect for WebSocket streams. Use updateLastBarFast() for high-frequency updates.

Get Context

const ctx = chart.getContext()

Get structured JSON with chart ID, viewport, visible bars, price ranges, and indicator data. The data source for your AI co-pilot. Controllable exposure with one flag.

Execute Commands

chart.execute({ type: 'fitContent' })

Universal command interface for zoom, pan, scale changes, and more. Type-safe and easy to use.
